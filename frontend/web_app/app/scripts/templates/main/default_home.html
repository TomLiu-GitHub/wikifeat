Welcome to Wikifeat

Looks like you've managed to install Wikifeat successfully. So you've got that going for you, which is nice.

This is the default start page for Wikifeat. You can (and should) designate one of your wiki pages as the site homepage.

What next?

  • Create a Wiki
  • Create some Users
  • Designate a Homepage

After you've done all that, you might want to peruse the documentation available on wikifeat.com.
